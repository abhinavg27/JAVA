Hibernate property values are: create, update, create-drop, validate and none:
props.setProperty("hibernate.ddl-auto", "create");
create â€“ Hibernate first drops existing tables, then creates new tables

props.setProperty("hibernate.ddl-auto", "update");
update â€“ uses existing table or creates a new one if not available; alter table if required.

validate â€“ Hibernate only validates whether the tables and columns exist, otherwise it throws an exception
none â€“ this value effectively turns off the DDL generation
